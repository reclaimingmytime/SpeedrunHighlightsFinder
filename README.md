## Description
A NestJS web app that filters the MCSR Ranked API for matches with at least one death, and displays links to the relevant Twitch VOD timestamps. Also includes links to dragon perch races where both players killed the dragon. The goal is to learn from mistakes and to find entertaining gameplay.

Requires _Node.js_ and _npm_.
## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
